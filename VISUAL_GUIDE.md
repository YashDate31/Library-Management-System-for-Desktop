## 🔄 Complete Workflow Diagram

```
┌─────────────────────────────────────────────────────┐
│          LIBRARY MANAGEMENT WORKFLOW                 │
└─────────────────────────────────────────────────────┘

    📚 BOOK MANAGEMENT
         │
         ├─ Add Book
         │  └─ ✅ Auto-generate ID (1, 2, 3...)
         │
         ├─ Issue Book
         │  └─ 📅 Tagged with academic year
         │
         └─ Return Book
            └─ 💰 Calculate fine if late

    👨‍🎓 STUDENT MANAGEMENT
         │
         ├─ Add Student
         │  └─ ✅ Appears at top (newest first)
         │
         └─ Promote Students (June)
            └─ 🎓 Creates new academic year

    📊 RECORDS & REPORTS
         │
         ├─ View Records
         │  ├─ Filter by Type
         │  ├─ Filter by Academic Year
         │  └─ 📅 Quick filters (7/15/30 days)
         │
         ├─ Find Overdue
         │  └─ 🟡 Yellow highlighted
         │
         └─ Send Letter
            └─ 🖱️ Double-click → 📄 Word doc

    🎓 YEAR-END PROCESS
         │
         ├─ June: End of academic year
         │
         ├─ Promote Students
         │  ├─ Enter letter number
         │  ├─ Confirm academic year
         │  └─ Promote All
         │
         ├─ System Creates New Year
         │  ├─ Deactivates old year
         │  └─ Activates new year
         │
         └─ July: New academic year starts!
```
